# 0.1.2 — 2026-05-23

## Added

- **MCPB bundle support** — `manifest.json`, `.mcpbignore`, and `bundle`/`lint:packaging`/`audit:refresh`/`publish-mcp` scripts added; `bun run bundle` produces a `.mcpb` for one-click Claude Desktop install.

## Changed

- **`@cyanheads/mcp-ts-core`** `^0.9.1 → ^0.9.5`
- **`zod`** added as explicit dependency (`^4.4.3`; previously a transitive-only dep).
- **`series_not_found` error code** changed from `invalidParams` to `notFound` — callers receive `JsonRpcErrorCode.NotFound` (-32001) instead of `InvalidParams` (-32602).
- **`no_data_for_period` error code** changed from `invalidParams` to `validationError` — callers receive `JsonRpcErrorCode.ValidationError` (-32003) instead of `InvalidParams` (-32602).
- **`calculations_not_supported` error code** changed from `invalidParams` to `validationError` — same code reclassification as `no_data_for_period`.
